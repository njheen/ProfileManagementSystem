Brief Project Description : 
This project is a web-based application designed to facilitate the effective management of student data within an academic environment.
The system allows users to register, view, search, and update student profiles through a modern, responsive interface.

List of Implemented Features :
- database integration (JDBC)
- Form fill and submission
- Display the submitted profile information after it has been successfully saved.
- Display all profile records stored in the database.
- search student profile using student id or name
- edit saved student profile
- delete saved student profile
- filter saved student profile record based on program
- interactive and appealing user-interface design
